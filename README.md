# Projeto Multi-App
Esse é um projeto desenvolvido em aula para nota da P1 do curso DSM
O app consiste em uma junção de 4 outros apps:
- Loteria: Um app que sorteia 6 numeros igual a mega-sena
- Calculadora IMC: Calcula o IMC e de acordo com o resultado ele manda uma activity diferente
- JoKenPo: O jogo classivo com 5 tentativas e o ganhador no final.
- Felizometro: Usando checkbox e radio pra opções e calcular o seu nivel de felicidade.
